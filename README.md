# 树树维护的卫月插件库

## 插件列表

| 插件名称 | 插件描述 |
|---------|---------|
| FF Logs Viewer (汉化) | 在游戏内查看 FF Logs 排名百分比，并在右键菜单中打开玩家日志。布局、样式、统计指标和默认度量等可在设置中自定义（仅支持在线安装） |
| Distance (汉化) | 在游戏内显示到目标/单位的距离，并可显示支持 Boss 的仇恨范围，支持姓名板与多种自定义显示（仅支持在线安装） |
| StarlightBreaker (汉化) | 国服屏蔽词解除：去除聊天过滤，保持原生交流，支持自定义白名单/黑名单（仅支持在线安装） |
| Item Vendor Location (汉化) | 查找物品的商店售卖位置，右键菜单显示所有可能商店与坐标，支持聊天命令快速查询（仅支持在线安装） |
| Enemy List Debuffs (汉化) | 在仇恨列表显示你的减益图标与倒计时，便于多目标场景追踪 DoT（仅支持在线安装） |
| Flash On Tell (汉化) | 收到悄悄话时让 FFXIV 窗口在任务栏闪烁，便于切回游戏（仅支持在线安装） |
| BigPlayerDebuffs (汉化) | 在目标状态栏放大由你施加的增益/减益图标，便于观察（仅支持在线安装） |
| Tilted (汉化) | 根据条件自动调整第三人称镜头角度与距离，支持战斗、副本、拔刀、骑乘、飞行、镜头靠近等模式（仅支持在线安装） |
| Waymark Preset Plugin (汉化) | 保存/编辑/放置/导入/导出与预览无限数量的场标预设，支持与游戏窗口联动（仅支持在线安装） |
