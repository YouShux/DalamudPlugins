# 树树维护的卫月插件库

## 插件列表

| 插件名称 | 插件描述 |
|---------|---------|
| FF Logs Viewer (汉化) | 在游戏内查看 FF Logs 排名百分比，并在右键菜单中打开玩家日志。布局、样式、统计指标和默认度量等可在设置中自定义。 |
| Distance (汉化) | 在游戏内显示到目标/单位的距离，并可显示支持 Boss 的仇恨范围，支持姓名板与多种自定义显示。 |
| StarlightBreaker (汉化) | 国服屏蔽词解除：去除聊天过滤，保持原生交流，支持自定义白名单/黑名单。 |
